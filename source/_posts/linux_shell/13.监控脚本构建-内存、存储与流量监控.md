## 进阶：内存、存储与流量监控菜单实现

**本节课目标：** 学习多列文本处理和数组统计，实现内存占用分析、磁盘 Inode 检测及实时流量监控。

### 核心工具：free & df & ifconfig
- **free**：查看内存及交换分区使用情况。
- **df -i**：查看文件系统 Inode 利用率（防止文件数过多导致磁盘虽有空间但无法写入）。
- **ifconfig / ip -s link**：获取网卡收发包及流量统计。

### AWK 高级扩展
- **AWK 数组**：用于分类统计。
    - `shells[$NF]++`：对最后一列统计出现次数。
- **AWK for 循环**：遍历统计后的数组结果。
- **标准化日志输出**：在本节的高级模块中，我们将继续使用 `log_info` 和 `log_error` 来规范化输出。

### 课程实践：高级模块实现

#### mem_use：内存占用分析

**创建 mem_use 菜单**，并使用 `log_info` 展示解析后的内存状态：

```bash
mem_use)
    # 内存利用率分析
    echo "  ---------------------------------------------------"
    # 使用 NR==2 定位第二行，并将 MB 转换为 G (保留一位小数)
    MEM_TOTAL=$(free -m | awk 'NR==2{printf "%.1fG\n", $2/1024}')
    USE=$(free -m | awk 'NR==2{printf "%.1fG\n", $3/1024}')
    FREE=$(free -m | awk 'NR==2{printf "%.1fG\n", $4/1024}')
    CACHE=$(free -m | awk 'NR==2{printf "%.1fG\n", ($6+$7)/1024}')
    
    log_info "内存统计结果:"
    echo -e "    总内存: $MEM_TOTAL"
    echo -e "    已使用: $USE"
    echo -e "    空闲中: $FREE"
    echo -e "    缓存中: $CACHE"
    echo "  -----------------------------------------------------"
    break ;;
```

> **扩展**：以上示例是使用 `NR==2` (行号) 准确定位 `free -m` 输出的第二行进行解析。掌握后，可练习改用 `/Mem/` (字符匹配) 实现更具通用性的脚本。


#### disk_inode：磁盘 Inode 检测

**创建 disk_inode 菜单**，并结合 `log_info` 和 `log_error`（或警告格式）展示状态：

```bash
disk_inode)
    # 1. 获取并输出所有分区的 Inode 利用率
    log_info "正在读取所有分区 Inode 利用率..."
    df -i | awk '/^\/dev/{print "    " $6 " = " $5}'

    # 2. 进阶：过滤利用率超过 90% 的分区并报警
    log_info "正在检测异常分区 (阈值 >90%)..."
    EX_DISK=$(df -i | awk '/^\/dev/ && int($5) > 90 {print $6 "("$5")"}')
    if [ -n "$EX_DISK" ]; then
        for disk in $EX_DISK; do
            log_error "分区 $disk Inode 利用率过高，请及时清理！"
        done
    else
        log_info "所有分区 Inode 状态正常。"
    fi
    break ;;
```

#### traffic：实时流量统计

```bash
traffic)
    # 1. 交互式网卡校验
    while true; do
        read -p "请输入欲监控的网卡名称 (如 eth0 或 em1): " eth
        if [ "$(ifconfig | grep -c "\<$eth\>")" -eq 1 ]; then
            log_info "网卡 $eth 校验成功，开始采样..."
            break
        else
            log_error "网卡不存在，请重新输入。"
        fi
    done

    # 2. 流量采样与计算
    echo "  ----------------------------------------------"
    echo -e " In (MB/s) ----------------------- Out (MB/s)"
    i=1
    while [[ $i -le 3 ]]; do
        # 兼容性解析
        OLD_IN=$(ifconfig "$eth" | awk -F'[: ]+' '/bytes/ && NR==8 {print $4} /bytes/ && NR==5 {print $6}')
        OLD_OUT=$(ifconfig "$eth" | awk -F'[: ]+' '/bytes/ && NR==8 {print $9} /bytes/ && NR==7 {print $6}')
        sleep 1
        NEW_IN=$(ifconfig "$eth" | awk -F'[: ]+' '/bytes/ && NR==8 {print $4} /bytes/ && NR==5 {print $6}')
        NEW_OUT=$(ifconfig "$eth" | awk -F'[: ]+' '/bytes/ && NR==8 {print $9} /bytes/ && NR==7 {print $6}')
        
        IN=$(( (NEW_IN - OLD_IN) / 1024 ))
        OUT=$(( (NEW_OUT - OLD_OUT) / 1024 ))
        
        echo "  [采样 ${i}] In: ${IN} KB/s | Out: ${OUT} KB/s"
        ((i++))
    done
    echo "  ------------------------------------------------------"
    break ;;
```
