## 交互式菜单与存储资源监控

**本节课目标：** 使用循环和选择语句构建交互菜单，集成 `iostat` 及磁盘空间监控分析。

### 1. 工具介绍：iostat & 磁盘命令
- **iostat**：监测磁盘 I/O 负载。
    - **常用参数**：`-x` (显示扩展统计信息)，`-k` (以 KB 为单位显示)。
    - **核心关注指标**：
        - `%util`：**磁盘忙碌百分比**。若该值接近 100%，表示磁盘接近满负荷。
        - `await`：**平均等待时间**（ms）。反映磁盘响应速度，值越大通常代表延迟越高。
        - `rkB/s` / `wkB/s`：每秒读取/写入的数据量。
    - **详细指标说明**:
        - `rrqm/s` / `wrqm/s`：每秒合并到设备队列的读/写请求数。
        - `r/s` / `w/s`：每秒完成的读/写次数（即 **IOPS**）。
        - `rMB/s` / `wMB/s`：每秒读/写的吞吐量（单位为 MB）。
        - `avgrq-sz`：平均每个 I/O 请求的大小（扇区数）。
        - `avgqu-sz`：平均 I/O 队列长度。
        - `r_await` / `w_await`：平均每个读/写操作的耗时（ms）。
- **df & fdisk**：查看文件系统硬盘利用率及真实磁盘总量。

### 2. Shell 核心语法
- **select in 循环**：用于生成交互式的编号菜单。
    - **基本语法**：
        ```bash
        select variable in list
        do
            commands
        done
        ```
    - **关键特性**：
        1. 它会自动显示菜单项并带上序号。
        2. 它会无限循环，直到在循环体中遇到 `break` 语句或按下 `Ctrl+D`。
        3. 通常与 `case in` 结合使用来处理用户选择。
    - **示例代码**：
        ```bash
        #!/bin/bash
        echo "请选择您的操作系统："
        select choice in "Linux" "Windows" "Mac OS" "Quit"
        do
            case $choice in
                "Linux")
                    echo "您选择了开源系统！"
                    ;;
                "Quit")
                    break
                    ;;
                *)
                    echo "无效选择"
                    ;;
            esac
        done
        ```
- **case in 语句**：根据用户输入执行不同操作。
- **while 循环**：用于让菜单持续运行或多次采样。
- **标准化日志输出**：为了保持与初始化脚本一致的风格，我们继续沿用上一节定义的日志函数：
    ```bash
    log_info() {
        echo -e "\033[32m[INFO] $(date '+%Y-%m-%d %H:%M:%S') - $1\033[0m"
    }
    log_error() {
        echo -e "\033[31m[ERROR] $(date '+%Y-%m-%d %H:%M:%S') - $1\033[0m"
    }
    ```

### 3. 课程实践：菜单构建与存储模块

#### CPU_LOAD 菜单构建

在获取监控指标后，使用 `log_info` 输出结果，增强信息的可读性。

```bash
# 模拟主循环
while true; do
    select input in cpu_load quit; do
        case $input in
            cpu_load)
                # 利用 vmstat 解析 CPU
                UTIL=$(vmstat | awk '{if(NR==3)print 100-$15"%"}')
                USER=$(vmstat |awk '{if(NR==3)print $13"%"}')
                log_info "总利用率: $UTIL"
                log_info "用户态使用率: $USER"
                break ;;
            quit) exit 0 ;;
        esac
    done
done
```

#### 优化输出格式与循环采样

通过 `log_info` 代替零散的 `echo`，使输出更加规范。

```bash
cpu_load)
    # CPU 利用率结果循环三次输出
    echo "-----------------------------------------------------"
    i=1
    while [[ $i -le 3 ]]; do
        log_info "正在进行第 ${i} 次采样..."
        UTIL=$(vmstat | awk '{if(NR==3)print 100-$15"%"}')
        USER=$(vmstat | awk '{if(NR==3)print $13"%"}')
        log_info "采样结果 -> 总利用率: $UTIL, 用户态: $USER"
        ((i++))
        sleep 1
    done
    echo "-----------------------------------------------------"
    break ;;
```

#### DISK_LOAD 菜单实现

集成 `iostat` 监控，并使用标准日志输出关键指标。

```bash
# 模拟主循环
while true; do
    select input in cpu_load disk_load quit; do
        case $input in
            cpu_load)
                # ... (同上) ...
                break ;;
            disk_load)
                # 使用 iostat 监控 I/O
                echo "-----------------------------------------------------"
                i=1
                while [[ $i -le 3 ]]; do
                    log_info "正在执行 I/O 采样 ${i}/3..."
                    UTIL=$(iostat -x -k |awk '/^[v|s]/{OFS=": ";print $1,$NF"%"}')
                    READ=$(iostat -x -k |awk '/^[v|s]/{OFS=": ";print $1,$6"KB"}')
                    IOWAIT=$(vmstat |awk '{if(NR==3)print $16"%"}')
                    
                    log_info "磁盘利用率:\n${UTIL}"
                    log_info "I/O 等待: $IOWAIT"
                    log_info "每秒读速率:\n$READ"
                    
                    ((i++))
                    sleep 1
                done
                echo "-----------------------------------------------------"
                break ;;
            quit) exit 0 ;;
            *) log_error "无效选择，请重新输入。" ;;
        esac
    done
done
```
